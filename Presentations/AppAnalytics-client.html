LAMP or WAMP

Use whichever works best for you, as long as it meets the requirements for Piwik!

  • VMWare Workstation for testing locally is handy.
  • Grab a virtual appliance - BitNami Stack for Piwik
    • http://bitnami.com/stack/piwik
  • Production host - anything from simple shared hosting to high availablitly cloud providers.
  • Highly recommended -

Tracking Client

  • Javascript Tracking tag, Image tracking, or using the API
  • REST API Clients - PHP, Java, Python C#, etc.
    • http://piwik.org/docs/tracking-api/
  • Official Piwik C# Tracking API client from GitHub - https://github.com/piwik/piwik-dotnet-tracker
  • Unmanaged Exports (DllExport for .Net) - https://www.nuget.org/packages/UnmanagedExports
